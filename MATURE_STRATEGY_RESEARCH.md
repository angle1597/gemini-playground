# A股量化策略研究报告

## 发现的成熟开源项目

### 1. a-share-quant-selector (116 stars)
**核心策略**: 碗口反弹策略

**选股条件**:
1. 上升趋势 - 知行短期趋势线 > 知行多空线
2. 异动放量阳线 - 近期存在放量阳线（成交量>=前日*N倍）
3. KDJ低位 - J值 <= 阈值，处于超卖区域
4. 剔除最大阴量 - 回顾期内成交量最大的一天如果是阴线，直接剔除

**技术指标**:
- 知行短期趋势线 = EMA(EMA(CLOSE, 10), 10)
- 知行多空线 = (MA14 + MA28 + MA57 + MA114) / 4

**优势**: 有完整的B1完美图形匹配，基于10个历史成功案例

## 策略优化方案

### 方案1: 直接使用成熟策略
- 克隆 a-share-quant-selector
- 使用碗口反弹策略
- 参数调优

### 方案2: 多策略组合
- 使用 KHunter 的多策略
- 策略OR组合（任一策略触发即入选）
- 策略AND组合（多策略共振）

### 方案3: 自研 + 借鉴
- 借鉴碗口反弹的技术指标
- 加入基本面筛选
- 加入大盘过滤

## 关键学习

**为什么成熟策略胜率高？**

1. **多维度验证** - 不只看技术指标，还看形态、量能、趋势
2. **风险过滤** - 自动剔除ST、退市、异常股
3. **分类标记** - 区分不同类型的买入信号
4. **参数调优** - 经过大量回测验证

**我们之前策略的问题**:
- 只看单一指标（均线+量比）
- 没有形态识别
- 没有风险过滤
- 参数未经验证
